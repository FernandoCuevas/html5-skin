PackageOO.Pulse
Classpublic class PulseFlashAdPlayer
InheritancePulseFlashAdPlayer flash.display.Sprite
ImplementsIAdPlaybackListener, IAdPlayer

Pulse Flash ad player. It can play VPAID and linear video ads. It does NOT do any tracking, only playback. Events are fired to Javascript when the player's state changes. The following flash vars are required : - playerId : a unique string that will allow the Javascript code to identify the player when events are fired. - jsCallback : function to pass the flash events to.
